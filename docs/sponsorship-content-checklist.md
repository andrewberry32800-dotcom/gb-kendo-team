# Sponsorship Page — Content Needed

Everything marked `X` on /sponsorship is a placeholder. This is the list of
real content needed to launch the page properly.

## 1. Numbers for "The Opportunity" (the sales pitch)

- [ ] Number of squad members and squads
- [ ] Championships / countries competed in (last 5–10 years)
- [ ] Total EKC / WKC medals
- [ ] Social media follower counts (all platforms combined)
- [ ] Website traffic if known (monthly visitors)
- [ ] Any press/broadcast coverage (BBC, local press, kendo media)

## 2. Sponsorship packages

- [ ] Decide the tiers (current draft: Gold / Silver / Bronze)
- [ ] Real prices per tier per season
- [ ] Confirm what each tier actually gets — the drafted benefits
      (logo on kit, social features, event invites) need sign-off from
      squad management and the BKA (kit branding rules!)
- [ ] Who handles the money — BKA account? Separate squad fund?
      Any charity/CASC status that affects Gift Aid?

## 3. Sponsor pack (PDF)

The page promises a downloadable pack. It needs:

- [ ] 4–8 page PDF: who we are, results, audience numbers, packages, contact
- [ ] High-res team photos (action + podium shots)
- [ ] A named contact person for sponsorship enquiries (name, role, email,
      phone)

## 4. Other ways to support

- [ ] Donation mechanism — JustGiving / GoFundMe page, bank transfer
      details, or a payment provider (Stripe/PayPal) if taking donations
      on-site
- [ ] List of real fundraising events with dates (charity taikai, seminars)
- [ ] Specific sponsorable costs with real figures (e.g. "£1,200 flies one
      athlete to the WKC")

## 5. Partners wall

- [ ] Logos (PNG/SVG, transparent background) from every current sponsor
- [ ] Written permission to display each logo
- [ ] Link URL for each sponsor

## 6. Legal / admin

- [ ] Sponsorship agreement template (even one page)
- [ ] VAT position on sponsorship income (sponsorship is usually a taxable
      supply — check with the BKA treasurer)
- [ ] Photo consent for any athletes featured in sponsor material,
      especially juniors (safeguarding)

## Quick wins to launch with

A minimal viable version needs only: real stats (section 1), one tier with a
real price, a named contact email, and one donation link. Everything else
can be added as it's confirmed.
